# REST API
  
URI와 HTTP를 기반으로 주로 Json형식을 사용함. 각 요청이 어떤 정보나 동작을 위하는 것인지 모습 자체만으로 추론 가능.

해당 정보의 상태정보를 공유함(get, post, put, delete  + patch)

HTTP요청시 어떤 URI에 어떤 mehtod를 사용할지에 대한 개발자들 사이에서의 약속

## swagger-ui
